# Image Requirements for GrowGuardians

Please add the following images to the `public` folder to complete the frontend:

## Required Images

### 1. logo.png
- **Location**: `public/logo.png`
- **Dimensions**: 242px × 211px
- **Usage**: Splash screen logo
- **Description**: The main GrowGuardians logo that appears on the splash screen

### 2. registration-illustration.png
- **Location**: `public/registration-illustration.png`
- **Dimensions**: 340px × 280px
- **Usage**: Registration screen illustration
- **Description**: An illustration showing a hand holding a smartphone with soft teal, green, and blue tones

### 3. loginpagepic.png
- **Location**: `public/loginpagepic.png`
- **Dimensions**: 300px × 300px
- **Usage**: Login screen illustration
- **Description**: An illustration conveying a welcoming, productivity-focused theme

## How to Add Images

1. Prepare your images with the exact dimensions specified above
2. Save them with the exact file names mentioned
3. Place them in the `frontend/public` folder
4. Restart the development server if it's already running

## Temporary Solution

If you don't have the images ready yet, you can:
1. Use the placeholder images provided
2. Use the logo images from your GrowGuardians branding
3. Find free illustrations from:
   - https://undraw.co
   - https://www.freepik.com
   - https://www.flaticon.com

## Note

The application will show broken image icons if these images are not present, but the functionality will work fine. Add the images when ready to see the complete design.
